# nuxeo-bulk-metadata-edit

-requires Nuxeo 10.10

Build with:

```
mvn clean install
```
and install:

```
./nuxeoctl mp-install nuxeo-bulk-metadata-edit/nuxeo-bulk-metadata-edit-package/target/nuxeo-bulk-metadata-edit-package-1.0-SNAPSHOT.zip  
```
